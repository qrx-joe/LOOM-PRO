## 🚀 使用指南

### 1. 启动应用

```bash
# 前端
cd frontend
npm run dev

# 后端
cd backend
npm run start:dev
```

### 2. 访问工作台

访问 `http://localhost:5174/workflow/new` 查看优化后的工作台

### 3. 主要功能

#### 创建工作流

1. 从左侧组件库拖拽节点到画布
2. 连接节点创建流程
3. 点击节点在右侧配置属性

#### 调试工作流

1. 点击顶部"调试"按钮
2. 输入测试数据
3. 点击"运行测试"
4. 查看执行日志和结果

#### 使用 HTTP 节点

1. 拖拽"HTTP 请求"节点到画布
2. 配置请求方法、URL、请求头等
3. 连接到其他节点
4. 运行测试查看结果
