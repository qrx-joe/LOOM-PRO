## 项目简介

AgentFlow Studio 是一个轻量级的类 Coze 智能体平台，支持：

- **可视化工作流编排**：拖拽式节点设计，支持条件分支
- **知识库与RAG检索**：文档上传、向量存储、混合检索
- **智能问答对话**：流式输出、上下文管理

## 技术栈

| 层级   | 技术                                   |
| ------ | -------------------------------------- |
| 前端   | Vue 3 + Vite + Element Plus + Vue Flow |
| 后端   | NestJS + TypeORM                       |
| 数据库 | PostgreSQL + pgvector                  |
| 缓存   | Redis                                  |
| AI     | OpenAI API / DeepSeek API              |
