## ⚠️ 已知问题

### 1. 伪向量导致检索失效

**问题**：使用伪向量（SKIP_EMBEDDING_API=true），向量相似度搜索返回空结果

**影响**：知识库检索功能无法正常工作

**解决方案**：

1. 配置真实的 Embedding API
2. 修改 `.env` 文件：
   ```env
   SKIP_EMBEDDING_API=false
   OPENAI_API_KEY=your_api_key
   EMBEDDING_MODEL=text-embedding-3-small
   ```

### 2. 流式输出前端显示

**问题**：流式输出后端正常，但前端可能没有实时更新

**状态**：已添加调试日志，需要在浏览器中测试

**解决方案**：

1. 打开浏览器开发者工具
2. 查看 Console 日志
3. 确认 SSE 数据接收和 Vue 响应式更新
